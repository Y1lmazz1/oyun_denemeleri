
 # oyun_denemeleri 🕹️

Bu repo, oyun geliştirme dünyasına attığım ilk adımları ve öğrendiğim temel mekanikleri denediğim küçük bir başlangıç projesidir.

## 📌 Neler Var?
* Temel hareket mantığı denemeleri.
* Öğrenme sürecindeki kod karalamaları.
* Basit oyun döngüsü çalışmaları.

## 🚀 Amaç
Oyun mantığını anlamak ve kod yazma pratiği yapmak için oluşturulmuş bir "playground" (oyun alanı) projesidir.
